WHAT IS NO-CODE

No-code is a software development approach that allows people with little or no programming experience to build applications.

Bubble IO is a no-code web development tool that allows individuals to build complex web applications without writing any code.

Look at some web apps built with bubble io

WEB DEVELOPMENT CURRICULUM

We have an online project based curriculum that spans 4 weeks where you will be learning how to build web applications from start to finish. At the end of each week you will have an assignment where you get to practice what you learnt on your own.

WEEK ONE

On week one we will take you through what no-code is and what you can build with no-code tools as well as careers with no-code. We then introduce you to web development by building a landing page.

WEEK TWO

On week two we introduce you to more complex concepts of web development by building a fully fledged chat application like WhatsApp. You will learn about databases, authentication and workflows.

WEEK THREE

On week three we teach you how to connect your app with other 3rd party apps using APIs and plugins. We do this by building a netflix like clone that will allow you to track your favorite tv shows.

WEEK FOUR

On week four we bring all we learned together by building a fully functional e-commerce application. We will introduce multi-user applications and how to handle and manage inventory.
